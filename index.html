Goal of Our Project

Inspired by one of our group member, who was once an amateur pingpong player,
we thought that it would be nice to create a robot which could collect all the
pingpong balls used during practise so that we don't have to go and pick all of
them up one by one. It could alleviate the physical burden on their back.

Although it sounded a bit ridiculous, as we discussed about the idea, we think that
it is actually feasible for us to make a robot that could transport an object to a
specific location based on the knowledge we learnt in this class.

Therefore, we decided that we would code our robot so it could physically push
the specificed object to the destinatation, while avoiding the obstacles in the
working environment.

It is an interesting project!

It is interesting because it could be implemented in real world situations and solve different kinds of problems. For example, it could be used as a cleaning robot on the street. Also, it could also be used to tidy up our room too! Its potential is endless.
